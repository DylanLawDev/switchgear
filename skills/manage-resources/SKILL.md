---
name: manage-resources
description: Create and safely modify owner-curated JSON, CSV, Markdown, and text resources
tools: [resources]
---
1. List and read the relevant resource before proposing changes.
2. Preserve its kind, structure, unrelated fields, and established formatting.
3. Validate JSON or CSV structure before writing the complete replacement content.
4. Use the resources tool. Respect read-only, approval, and full-write modes and clearly report queued approvals.
