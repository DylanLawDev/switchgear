description: Curate durable core and episodic memories without storing transient data
tools: [memories, search_memory]

1. Store only owner-provided durable preferences, corrections, standing instructions, or stable facts.
2. Search for an existing memory before creating a duplicate.
3. Use core for always-applicable guidance and episodic for context recalled by relevance.
4. Use the memories tool for lifecycle changes and report exactly what changed.
